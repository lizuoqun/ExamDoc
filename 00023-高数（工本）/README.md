### 函数

- 函数四大特性
    - 有界性
    - 单调性
        - 证明在区间内单调递增，设存在x1,x2(x1<x2)得到f(x2)-f(x1)>0
    - 奇偶性
        - 奇函数：f(x) = -f(x)
        - 偶函数：f(x) = f(-x)
        - 两个偶函数之和、之积都为偶函数
        - 两个奇函数的和为偶函数、之积为奇函数
        - 奇函数和偶函数之积为奇函数
    - 周期性
        - 存在常量T，存在f(x) = f(x+T)，T为周期
- 反函数
    - y = x + 1 的反函数为 x = y - 1（图像关于y=x对称）
- 复合函数
    - y=1/u-3，u= √(2x-1)，那么y = 1/(√(2x-1) -3) 求定义域即 2x-1 > 0 和 √(2x-1) -3 ≠ 0
- 初等函数
    - 幂函数 y = x^n（n=-1（1/x）,1/2(根号x),1,2,3）
    - > a^0 = 1(a≠0)
      >
      > a^-n = 1/a^n(a≠0,n属于N)
      >
      > a^(m/n) = n√(a^m) (a>=0，mn属于N，mn互质)
    - 指数函数 y = a^x，a > 0，常见的有 y=e^x，y=2^x，y=10^x
    - 对数函数 y = log a(x) (a>0且a≠1)
    - > 对数 log a(N) = b <=> a^b = N（零和负数没有对数）a^(log a(N)) = N

    - 三角函数
    - sin等于对边/斜边、cos等于邻边/斜边、tan等于对边/邻边（求角度值直接用1、2、√3加上等腰直角三角形算45度）

### 数列

- 等比数列
    - 通项公式 an = a1 * q^(n-1)
    - 中项 b^2 = ac

### 导数

- 简述
    - 求导 f(x) = x^2 求导
        - > x->0 f'(x) = [f(x + △x) - f(x)] / △x = [x^2 + 2x * △x - x^2] / △x = 2x
    - 求导的结果为切线的斜率
- 初等函数的导数【计算和上面一样】
    - 常值函数 f(x) = c，导数为0
    - 幂函数 f(x) = x^n，导数为nx^(n-1)
    - 三角函数 f(x) = sin x，导数为cos x
    - 三角函数 f(x) = cos x，导数为-sin x
    - 三角函数 f(x) = tan x，导数为sec^2 x
    - 三角函数 f(x) = cot x，导数为-csc^2 x
    - 三角函数 f(x) = arc sin x，导数为1/√1-x^2
    - 三角函数 f(x) = arc cos x，导数为-1/√1-x^2
    - 三角函数 f(x) = arc tan x，导数为1/(1+x^2)
    - 三角函数 f(x) = arc cot x，导数为-1/(1+x^2)
    - 指数函数 f(x) = e^x，导数为e^x
    - 对数函数 f(x) = ln x，导数为1/x
- 导数的四则运算
    - 加减法 [u(x)+v(x)]' = u'(x) + v'(x)
    - 乘法 [u(x)v(x)]' = u(x)v'(x) + u'(x)v(x)
    - 除法 [u(x)/v(x)]' = [u(x)v'(x) - u'(x)v(x)] / v^2(x)
    - 复合函数求导：例：f(x) = sin(x^2)
        - > dy/dx = dy/du * du/dx 【y=sinu、u=x^2】 = cosu * 2x = cos(x^2) * 2x
- 特殊函数求导
    - 幂指函数求导 y = x^x (x>0)
        - 两边各取ln：ln y = x * ln x，两边求导，ln y是一个复合函数，求导为 1/y * y'，右边为1 * ln x + x *
          1/x = ln x + 1。那就是 1/y * y' = ln x + 1。即 y' = y * (ln x + 1) = x^x * (ln x + 1)
- 高阶函数求导
    - y = x * e^x , y的n阶导数为：y = (x+n) * e^x

### 导数应用

洛必达法则

- 存在两个函数f(x)和g(x)
    - lim x->x0 f(x) = g(x) = 0 或 ∞
    - 在a附近两个函数皆可导
    - lim x->x0 f'(x) / g'(x)存在或者为 ∞
- 则lim x->x0 f(x) / g(x) = lim x->x0 f'(x) / g'(x)

函数单调性

- 对f(x)求导，若f'(x) > 0，则f(x)单调递增，若f'(x) < 0，则f(x)单调递减，若f'(x) = 0，则f(x)无单调性
