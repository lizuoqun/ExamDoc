## 等价演算

- 双重否定 p = ¬(¬p)
- 交换 p v q = q v p
- 结合 (p ∧ q) ∧ r = p ∧ (q ∧ r)
- 德摩根 ¬(p ∧ q) = ¬p ∨ ¬q 或 ¬(p ∨ q) = ¬p ∧ ¬q
- 分配 p ∨ (q ∧ r) = (p ∨ q) ∧ (p ∨ r) 或 p ∧ (q ∨ r) = (p ∧ q) ∨ (p ∧ r)
- 吸收 p ∨ (p ∧ q) = p 或 p ∧ (p ∨ q) = p
- 条件等价 p → q = ¬p ∨ q
- 双条件等价 p ↔ q = (p → q) ∧ (q → p)
- 假言 p → q = ¬q → ¬p
